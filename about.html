What is EMERGIAI?

EMERGIAI is an AI-powered emergency response assistant that provides immediate guidance, triage, and nearby help during medical, fire, and accident emergencies.

Problem It Solves

In a crisis, people need quick, clear instructions and nearby resources. EMERGIAI assists with first-aid guidance, severity assessment, and directions to hospitals or essential services.

Technologies Used

  • Backend: Python FastAPI
  • AI: Rule-based triage + optional OpenAI LLM
  • Frontend: HTML, Tailwind CSS, Vanilla JS
  • Maps: OpenStreetMap Overpass
